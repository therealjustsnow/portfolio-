Self-Hosting Guide

1. Clone

git clone https://github.com/therealjustsnow/NanoBot.git
cd NanoBot

2. Install

python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt

Required dependencies:

  • discord.py>=2.3.0
  • aiosqlite>=0.19.0
  • aiohttp>=3.9.0
  • psutil>=5.9.0
  • Pillow>=10.0.0

3. Configure

  1. Copy example_config.ini to config.ini.
  2. Set [bot] token.
  3. Optional: set [bot] owner_id for owner commands.
  4. Optional: set [groq] groq_api_key for /eli5.
  5. Optional: fill list/webhook tokens in [votes].
export DISCORD_TOKEN=your_token_here

4. Discord Portal

  • Enable Server Members Intent.
  • Enable Message Content Intent.

5. Optional migration

python migrate.py

6. Run

python run.py
# or
python main.py

Troubleshooting

  • Prefix commands missing: check Message Content Intent.
  • Permission errors: check bot role permissions + role hierarchy.
  • Slash commands stale: run owner command n!sync.
  • Startup failures: inspect logs/nanobot.log and config.ini.

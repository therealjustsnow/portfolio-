Mobile-first notes

Source-backed reasons NanoBot fits phone moderation

  • Many moderation commands can target last sender if user field is blank.
  • Role panels are button-driven and persist across restarts.
  • Tags support shorthand like n!tag hello and direct n!hello.
  • Reminder, fun, and utility flows are built around short command inputs.

📱 No More Copying IDs

The biggest pain of mobile moderation: grabbing user IDs. NanoBot removes it entirely. Every mod command auto-targets the last person who sent a message in the channel. See someone break the rules? Just type the command. No selecting, no scrolling, no copy-paste.

/last — confirm the current target anytime
/ban Slash
n!ban Prefix
@NanoBot ban Mention
🔨
Complete Moderation

Ban, tempban, clean ban, softban, massban. Kick, timeout, unfreeze. Lock channels, set slowmode with auto-disable. Purge with filters. Move voice users. All timed actions survive restarts.

⚠️
Warning System

Issue, track, and clear warnings per user. Set auto-kick and auto-ban thresholds so repeat offenders are handled automatically. Pair with private mod notes for full internal tracking.

🛡️
AutoMod

Spam, invite links, external links, caps, mass mentions, bad words, custom regex. Each rule can delete, warn, or timeout. Exempt roles and channels. Runs quietly in the background.

🏷️
Tags

Save messages and reuse them instantly. Personal or server-wide. Supports text and images. Fire any tag with n!tagname — no menus, no extra steps.

🎭
Role Panels

Button-based role menus that work great on mobile. Tap to toggle. Single-select mode available. Quick generators for colors, pronouns, age ranges, and regions.

📋
Audit Log

Message edits and deletes, joins, leaves, bans, unbans, nickname changes, role updates, channel changes. Toggle each event individually. One clean feed.

⏰
Reminders

One-time reminders with natural language. Recurring reminders that repeat on a schedule, like calendar events but in Discord. All survive restarts.

🎉
Fun & Community

59 social and reaction GIF commands, compatibility shipping, 8-ball, Would You Rather, FML stories, anime images, and ELI5 powered by AI. Because even mods need a break.

👋

Welcome & Leave Messages

Custom join and leave messages with variables like {user} {mention} {server} {count}. Supports embeds and images. Test everything before enabling.

🔍

Clean Info Commands

User cards, server info, role breakdowns, avatars, and banners. All formatted to read well on a phone screen.

💾

Persistent & Reliable

Every timed ban, timeout, slowmode, warning, and setting survives a restart. Nothing gets lost in a crash. Your moderation state is always intact.
